## AIM  
To develop a Python program using OpenCV to perform image handling and pixel transformations such as reading, displaying, brightness adjustment, contrast modification, and channel operations.  

## SOFTWARE REQUIRED  
- Anaconda (Python 3.7 or above)  
- Jupyter Notebook  
- OpenCV (cv2)  
- NumPy  
- Matplotlib  

## ALGORITHM  

### Step 1  
Read the input image using OpenCV and display it.  

### Step 2  
Convert the image into grayscale and obtain its properties (width, height, channels).  

### Step 3  
Display the image using Matplotlib.  

### Step 4  
Save the image in a different format (PNG).  

### Step 5  
Convert the image into color format and display it.  

### Step 6  
Crop a region of interest (ROI) from the image.  

### Step 7  
Resize the image and apply horizontal flipping.  

### Step 8  
Load another image and add text annotation.  

### Step 9  
Draw geometric shapes such as rectangles on the image.  

### Step 10  
Adjust image brightness using matrix operations.  

### Step 11  
Generate brighter and darker versions of the image.  

### Step 12  
Modify image contrast using scaling factors.  

### Step 13  
Split the image into B, G, R channels and display them.  

### Step 14  
Merge the channels back and display the image.  

### Step 15  
Convert the image into HSV format, split channels, and merge them again.  



## OUTPUT  
- Image successfully read and displayed  
- Brightness adjusted (brighter and darker images obtained)  
- Contrast modified using scaling  
- Image transformations (crop, resize, flip) performed  
- Channel splitting and merging completed  

## RESULT  
Thus, the Python program using OpenCV was successfully implemented to perform image handling and pixel transformations including brightness and contrast adjustments, annotation, and channel operations.
